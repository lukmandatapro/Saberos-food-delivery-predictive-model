# Sabores Ibericos Food Delivery Prediction

## Business Problem
The company experienced increasing delivery cost and longer courier waiting times across multiple cities.

## Objective
Identify the key factors contributing to high delivery costs and courier delays by analyzing  historical order-level data and build a predictive model

## Tools Used
- Python
- Pandas
- Scikit-learn
- SQL

## Analysis Performed
- Data cleaning
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Ffeature Engineering
- Machine Learning Model

## Key Insight
Distance from store to customer and size of the order are the biggest drivers of cost. Delivery fees and courier waiting time also matter but less so.

## Model
- Support Vector Classifiers
- Random Forest Classifiers
- Logistic regression
- Extreem gradient Boosting Classifiers
- Stochastic Gradient Descent Classifiers
- Gausian Naive Bayes
- Decision Tree classifiers
